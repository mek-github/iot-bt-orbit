# Assets Directory

This directory contains app assets like icons and splash screens.

To generate proper icons and splash screens, you can use:

```bash
npx expo-asset-utils generate-icons
```

Or manually create:
- `icon.png` - 1024x1024px app icon
- `splash.png` - 1242x2436px splash screen
- `adaptive-icon.png` - 1024x1024px Android adaptive icon
- `favicon.png` - 16x16px or 32x32px favicon

For now, the app will use default Expo assets.
